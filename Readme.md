Arquivos utilizados:
- sync_car.c: arquivo com a implementação das funções de sincronização
- sync_car.h: arquivo header com a declaração da struct station 
- sync_car-runner.c: arquivo de teste 

Como compilar:
$ make

Como executar:
$ ./sync_car

Como limpar os arquivos compilados:
$ make clean

Sistema operacional: Linux Ubuntu (WSL2)
